# Fantasy_Football_Visualizer
A way to visualize football player's statistics and fantasy football production with predictions

1) Open a Java App with fields for player name and stats by week
2) Indicate which week of the season it is
3) Create a new player
4) Add a player’s stats for all weeks up until present
5) Generate a graph of production for fantasy football
6) Generate predictions for the upcoming weeks for fantasy football on the graph
7) Change Fantasy football production values based on scoring rules for user
8) Bar graph of statistics over the weeks
9) Generate predictions for the upcoming weeks for statistics in the bar graph

Stretch Functionalities
10) Create additional players for side-by-side comparison for both graphs
11) Change graphic color depending on team input
